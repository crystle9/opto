这是基于 RubyOnRails 框架的个人博客。已实现的功能有：

1. 用户身份验证（使用 devise gem）
2. 管理（CRUD）用户自己的文章（使用 scaffold）
3. 新建用户的书，实质上是定义文章与文章之间的一种顺序

注意：此目录树少了/config/database.yml文件，请自行添加。
